# SINACOR-PARSER
Parser para notas de corretagem no formato SINACOR. 

## Utilização

1. Coloque suas notas de corretagem na pasta `pdf`
2. Abra um terminal e navegue até a raíz do repo
3. Rode o script `parser.py`
    ```
    > python .\parser.py
    ```
